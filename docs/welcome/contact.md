# Contact & Support

Need help with the textbook content or technical issues? We're here to assist you.

## Getting Help with Content

For questions about textbook content, use the **AI-powered chat assistant** located in the bottom-right corner of every page. The assistant can help with:

- Explaining complex concepts
- Clarifying code examples
- Providing additional resources
- Answering assessment questions

## Technical Support

For technical issues with the textbook platform:

- **GitHub Issues**: Report bugs or suggest improvements at [GitHub Repository](https://github.com/physical-ai-textbook/book-ai/issues)
- **Email**: textbook-support@physical-ai.org (response within 48 hours)
- **Community Forum**: Join discussions at [Physical AI Community](https://community.physical-ai.org)

## Feedback & Contributions

We welcome feedback and contributions:

- **Content Corrections**: Found an error? Report it via GitHub issues
- **Suggestions**: Have ideas for improvement? Share them on our forum
- **Contributions**: Interested in contributing? See our [Contributing Guide](https://github.com/physical-ai-textbook/book-ai/blob/main/CONTRIBUTING.md)

## Accessibility Support

For accessibility-related issues or special accommodations:

- **Accessibility Team**: accessibility@physical-ai.org
- **Screen Reader Support**: Contact for additional resources
- **Alternative Formats**: Available upon request

## Learning Support

For academic support with concepts:

- **Study Groups**: Join virtual study sessions
- **Office Hours**: Weekly Q&A sessions (schedule on forum)
- **Peer Mentoring**: Connect with advanced learners

## System Requirements

To access all features of this textbook:

- **Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Screen**: Minimum 1024x768 resolution
- **Bandwidth**: 5 Mbps recommended for video content
- **JavaScript**: Enabled for interactive features
